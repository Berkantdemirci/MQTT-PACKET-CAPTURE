# MQTT-Packet-Capture

## Preliminary

You need to install libpcap library to be able to use the project.

```sh
sudo apt-get install git libpcap-dev
```

## Compile & Run

Just run the "compile.sh" bash script to compile. The binary named "mqtt_listen" must be created.
